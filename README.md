IHS: Intelligent Home Sensing
=============================

Description
-----------
IHS is a project based on the [IoT][] to build a control and monitoring system for home using RF technology and master/multi-slave communication. 

[IoT]: http://en.wikipedia.org/wiki/Internet_of_Things

Usage and documentation
-----------------------
Please see the [wiki][] for specification and other documentation on IHS.
